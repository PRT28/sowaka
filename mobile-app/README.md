# Sowaka

Flutter application for Sowaka Connect.

## Run

Start the backend, then run one of these commands from this directory.

Android emulator:

```bash
flutter run
```

Web:

```bash
flutter run -d chrome --web-port=8080
```

Use a custom API URL for physical devices and deployed builds:

```bash
flutter run --dart-define=API_BASE_URL=https://api.example.com
```
